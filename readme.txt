Display Ramadan 2026 prayer times (Sehri & Iftar) for all divisions of Bangladesh in Bangla language with automatic time adjustments.

== Description ==

Ramadan Calendar Bangladesh is a comprehensive WordPress plugin that provides accurate Sehri and Iftar times for all major divisions in Bangladesh. The plugin displays times in Bangla (Bengali) language and automatically adjusts times based on geographical locations within Bangladesh.

= Key Features =

* Shows daily Sehri and Iftar times for 8 divisions of Bangladesh
* Complete Ramadan calendar for 2026
* Automatic time adjustments for different divisions
* Full Bangla (Bengali) language support
* Responsive design for all devices
* Easy to use shortcodes
* Automatic Eid Mubarak message after Ramadan
* Beautiful, clean interface
* SolaimanLipi font integration

= Available Divisions =

* ঢাকা (Dhaka)
* চট্টগ্রাম (Chattogram)
* সিলেট (Sylhet)
* রাজশাহী (Rajshahi)
* বরিশাল (Barishal)
* খুলনা (Khulna)
* রংপুর (Rangpur)
* ময়মনসিংহ (Mymensingh)

= Shortcodes =

1. Daily Timetable:
   `[mjashik_ramadan_time]`
   Shows today's Sehri and Iftar times for all divisions.

2. Full Calendar:
   `[mjashik_ramadan_calendar]`
   Displays complete Ramadan calendar for 2026.

= Usage =

1. Install and activate the plugin
2. Go to "Ramadan Time" in your WordPress admin menu
3. Copy the desired shortcode
4. Paste the shortcode in any post, page, or widget

= Perfect For =

* Mosques and Islamic centers in Bangladesh
* News websites covering Ramadan
* Community websites
* Personal blogs
* Any website catering to Bangladeshi Muslims

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/ramadan-calendar-bangladesh` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the shortcodes in your posts or pages
4. Access plugin settings via "Ramadan Time" menu in admin panel

== Frequently Asked Questions ==

= Are the times accurate? =

Yes, all times are based on standard calculations for Bangladesh with specific adjustments for each division.

= Can I use this plugin outside Bangladesh? =

The plugin is specifically designed for Bangladesh divisions. While it will work anywhere, the times are only accurate for Bangladesh.

= Do I need to update the times manually? =

No, all times are pre-calculated and automatically adjusted for each division.

= Will this work with any theme? =

Yes, the plugin is designed to work with any properly coded WordPress theme.

= Is the plugin free? =

Yes, this plugin is completely free and open source.

# Ramadan Calendar Bangladesh

A WordPress plugin to display Ramadan 2026 prayer times for Bangladesh divisions in Bangla language.

## Features

* Daily Sehri & Iftar times
* Full Ramadan calendar
* 8 division coverage
* Automatic time adjustments
* Bangla language support
* Responsive design
* Easy-to-use shortcodes
* Built-in Eid greeting

## Requirements

* WordPress 5.0 or higher
* PHP 7.2 or higher
* Any modern web browser

## Installation

1. Download the plugin
2. Upload to `/wp-content/plugins/`
3. Activate in WordPress admin
4. Use shortcodes:
   * `[mjashik_ramadan_time]` - Daily times
   * `[mjashik_ramadan_calendar]` - Full calendar
